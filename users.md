# USERS of the Water Sampler App and their responsibilities

## Administrator
### Email: admin@watersampler.com
### Role: Admin
    -Full access to the system

    -Manage user accounts and roles

    -Configure system settings

    -Define operator permissions

## Operator
### Email: operator@watersampler.com
### Role: Operator/Technician

    -Perform operational tasks
    - access features enabled by admin

## Viewer
### Email: viewer@watersampler.com
# Role: Viewer

    - Can view data and system information


## All newly registered users are created as "VIEWER" by default and user role upgrades are performed manually via Firestore
